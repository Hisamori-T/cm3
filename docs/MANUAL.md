## 1. ログイン・基本操作

### ログイン

1. ブラウザで `https://cmv3.fact-ally.com` を開く
2. 管理者から発行されたメールアドレスとパスワードを入力して **ログイン** をクリック

> **ブックマーク推奨**：スマートフォンでも同じURLにアクセスできます

### 画面の構成

```
┌─────────────────────────────────────────────────────┐
│  CL  Construction Mgr    ページ名    [アクションボタン] │  ← 上部バー
├──────────┬──────────────────────────────────────────┤
│ ダッシュ  │                                          │
│ 案件一覧  │           メインコンテンツ                │
│ カンバン  │                                          │
│ 工程表   │                                          │
│ 日報     │                                          │
│ カレンダー│                                          │
│ 業者/顧客│                                          │
│ ─設定─  │                                          │
│ ユーザー  │                                          │
│ Excel    │                                          │
│ 自社情報 │                                          │
└──────────┴──────────────────────────────────────────┘
```

- 左側のサイドバーでメニューを切り替えます
- 上部バー右端のボタンでページごとのアクション（新規作成・保存など）が実行できます

### ログアウト・プロフィール変更

- 画面左下のアバター（名前の頭文字）をクリック → **プロフィール** または **ログアウト**

## 2. ダッシュボード

**URL**：`/dashboard`

経営状況をひと目で把握するトップページです。

| カード | 内容 |
|---|---|
| 売上合計 | 今月の受注・請求合計金額 |
| 進行中案件数 | ステータス「着工〜施工中」の案件数 |
| 利益率 | 全案件の平均営業利益率 |
| ステータス分布 | 7段階ステータス別の案件数内訳 |
| 月別推移グラフ | 過去12ヶ月の売上推移 |
| 利益率ランキング | 案件別の利益率トップ10 |

## 3. 案件管理

### 3-1. 案件一覧

**URL**：`/projects`

- 工事番号・工事名・ステータス・担当者・工期をまとめて一覧表示
- 上部の検索ボックスで工事名・発注者名をキーワード検索可能
- ステータスタブでフィルタリング（全件 / 見積中 / 受注 / 着工 など）

### 3-2. 新規案件の作成

1. 案件一覧の右上 **＋ 新規作成** ボタンをクリック
2. 工事名・発注者・工事場所を入力
3. 工事番号は `{年}-{社員番号}-{連番}` で**自動採番**されます（手動変更可）
4. **作成** をクリック → 案件詳細画面へ移動

### 3-3. 案件詳細画面

案件を開くと上部にタブメニューが並びます。

| タブ | 内容 |
|---|---|
| **詳細** | 案件の基本情報・担当者・工期・ステータス |
| **QCDS** | 原価算定表（直接工事費・現場経費・利益逆算） |
| **業者見積** | 業者から届いた見積の管理 |
| **顧客見積** | 顧客向け見積書の作成・PDF出力 |
| **注文書** | 顧客への注文書作成 |
| **注文請書** | 顧客からの注文請書管理 |
| **請求書** | 請求書の作成・PDF出力 |
| **進捗** | 写真・図面・施工記録の投稿 |
| **工程表** | 案件内のガントチャート |
| **出面** | 作業員の出面記録 |
| **写真台帳** | 現場写真の整理・出力 |
| **発注書** | 業者への発注書作成 |
| **編集履歴** | 誰がいつ変更したかの全履歴 |

### 3-4. 案件情報の編集

1. 詳細タブを開く
2. 「**案件情報**」カードヘッダーの右端 **編集** ボタンをクリック
3. 各項目を修正 → **保存** ボタン（同じ位置に切り替わります）

> 担当者欄（「**担当者**」カード）にも同様に編集ボタンがあります

### 3-5. ステータス変更

詳細タブの上部にある 7段階のボタンをクリックするだけで変更できます。

```
見積中 → 受注 → 着工 → 施工中 → 完工 → 請求済 → 入金済
```

> 変更権限：管理者・マネージャー・作成者本人のみ

### 3-6. カンバンビュー

**URL**：`/projects/kanban`

ステータス別の案件カードをドラッグ＆ドロップで移動できます。全体の進捗状況を視覚的に把握するのに便利です。

## 4. QCDS（原価算定）

**URL**：`/projects/{id}/qcds`

クラップ独自の原価算定表です。

### 4-1. 主要タブ

| タブ | 内容 |
|---|---|
| 原価集計 | 直接工事費・現場経費・一般管理費の集計 |
| 実行予算 | 各工種の予算入力 |
| 取決見通 | 受注後の見通し修正 |
| 精算見通 | 完工後の精算 |
| 業務マスタ | 経費率・保険料率・人件費単価の設定 |

### 4-2. 利益逆算テーブル

工事原価から目標利益率を複数設定すると、顧客に提示すべき見積金額を自動計算します。

- **印紙税**も工事金額に応じて自動算定されます

## 5. 顧客見積書

**URL**：`/projects/{id}/quote`

### 5-1. 見積書の作成

1. 見積タブを開く → **＋ 新規見積** をクリック
2. 大項目（工種）と明細を入力
3. 値引き・消費税が自動計算されます

### 5-2. 承認スタンプ（電子捺印）

1. 見積書を開く → 右上の **承認** ボタン
2. 捺印種別（承認 / 確認 / 参照）を選択してクリック
3. 担当者の苗字がスタンプとして記録されます
4. 見積書を変更するとスタンプは**自動でリセット**されます（改ざん防止）

### 5-3. PDF・Excel出力

- 見積書詳細画面 右上 **Excel出力** または **PDF** ボタン
- PDF：3部構成（表紙 / 総括表 / 大項目別明細）

## 6. 業者見積スキャン（AI）

**URL**：`/scan`

業者から届いた紙・PDFの見積書をAI（Gemini）が読み取り、QCDS・見積書に自動転記します。

### 手順

1. **スキャン** メニューを開く
2. 業者見積のPDF / 画像ファイルをアップロード
3. AI解析が完了するまで待つ（数十秒〜2分程度）
4. 解析結果を**必ず確認**してから「QCDS/見積に反映」をクリック

> 解析結果は必ずユーザーがレビューしてから反映してください。数値の誤読が発生する場合があります。

## 7. 注文書・注文請書

**URL**：`/projects/{id}/order` / `/projects/{id}/acknowledgment`

### 注文書の作成

1. 注文書タブを開く
2. **案件から適用** ボタン（緑）をクリック → 案件情報・見積金額・工期が自動入力
3. 納期・支払条件を確認・修正
4. **保存** → **PDF出力** で取引先に送付

### 注文請書

受け取った注文請書を管理します。承認スタンプで受領確認を記録できます。

## 8. 請求書

**URL**：`/projects/{id}/invoice`

1. 請求書タブ → **＋ 新規請求** をクリック
2. 請求金額・消費税・支払期日を入力
3. **PDF出力** で取引先に送付

## 9. 進捗管理（写真・図面）

**URL**：`/projects/{id}/progress`

現場の施工状況を記録します。スマートフォンから直接投稿できます。

### 投稿方法

ページ内に3種類のボタンがあります：

| ボタン | 内容 |
|---|---|
| **テキスト** | 施工記録・コメントをテキストで投稿 |
| **写真** | スマホカメラで撮影または画像ファイルを選択してアップロード |
| **図面** | PDF・CADファイルなどをアップロード |

### 投稿の削除

各投稿の右上にある **×** ボタンで削除できます（権限チェックあり）。

## 11. カレンダー・担当者スケジュール

**URL**：`/calendar`

**「誰がいつどこの現場にいるか」を1画面で把握できます。**

### 画面の見方

カレンダーの各日付に4種類の色チップが表示されます：

| 色 | 種類 | 内容 |
|---|---|---|
| シアン（青緑） | **担当者スケジュール** | 現場訪問の予定（誰がどの現場へ） |
| 緑 | **日報（提出済）** | その日に提出された日報の現場名 |
| 紫 | 打合せ | 社内・客先との打合せ |
| 赤 | マイルストーン | 重要な節目（検査・引渡など） |
| 橙 | 業者訪問 | 業者との打合せ |
| グレー | その他 | 個人予定・有給など |

### 日付をタップすると日詳細パネルが開きます

右側に当日のイベントと日報が一覧表示されます。

### 担当者スケジュール（現場訪問予定）の登録

1. 登録したい日付をクリック → 日詳細パネルを開く
2. **＋ イベント追加** をクリック
3. 種別を **「現場」** に変更
4. 案件をドロップダウンから選択
5. 担当する社員にチェックを入れる（複数可）
6. 日時を設定 → **保存**

> 「現場」種別で登録すると、カレンダーにシアン色で表示されます

### イベントの削除

日詳細パネルのイベント一覧の右端 **×** をクリック

## 12. 日報

日報は2つの方法で入力できます。

### 方法①：カレンダーから入力（推奨）

1. `/calendar` を開く
2. 入力したい日付をクリック → 日詳細パネル
3. **＋ 日報を書く**（緑ボタン）をクリック
4. 天気・案件・作業内容・開始/終了時刻を入力
5. 複数の現場を掛け持ちした場合は **「＋ 案件を追加」** で行を追加
6. **提出する** をクリック

### 方法②：日報ページから入力

**URL**：`/daily-report`

1. 右上の **＋ 日報を書く** をクリック
2. 同様のフォームで入力・提出

### 日報の確認・修正

- 日報ページのタイムラインで過去の日報を確認
- 自分の日報（または管理者）は鉛筆アイコンから修正可能

## 14. 顧客マスタ

**URL**：`/clients`

取引先（発注者）の情報を管理します。

| 機能 | 内容 |
|---|---|
| 顧客登録 | 会社名・住所・電話番号などを登録 |
| 店舗管理 | 同一顧客の複数店舗を管理（例：平和堂〇〇店） |
| 担当者管理 | 顧客窓口の担当者情報を登録 |
| 過去案件 | 同一顧客の過去受注履歴を一覧で確認 |

### 顧客コードの自動生成

新規登録時に **自動生成** ボタンをクリックすると、会社名の先頭4文字＋タイムスタンプでコードが自動作成されます。

## 15. 管理者専用機能

> 以下の機能は **管理者（admin / super_admin）** のみ使用できます

### ユーザー管理（`/admin/users`）

| 操作 | 内容 |
|---|---|
| ユーザー追加 | メールアドレス・氏名・権限を設定して招待 |
| 権限変更 | 一般 / マネージャー / 会計担当 / 管理者 / 最高管理者 |
| パスワード変更 | 対象ユーザーの一時パスワードを設定 |

### Excelインポート（`/admin/import`）

既存のExcel工事台帳データを一括インポートします。インポートフォーマットは管理者に確認してください。

### 自社情報設定（`/admin/company`）

会社名・住所・電話番号・振込先などを設定します。設定内容は帳票（PDF・Excel）に自動反映されます。

## 17. 困ったときは

### よくある質問

**Q：ログインできない**  
A：管理者にパスワードの再設定を依頼してください。

**Q：案件が編集できない**  
A：編集権限は「管理者 / マネージャー / 案件の作成者本人」のみです。権限変更は管理者に依頼してください。

**Q：見積書のPDFが出力されない**  
A：ブラウザのポップアップブロックが有効な場合、PDFが開かれないことがあります。ポップアップを許可する設定をお試しください。

**Q：業者見積のAI解析が遅い / 失敗する**  
A：スキャン品質が低い画像は読み取り精度が落ちます。解像度300dpi以上のPDFをお使いください。解析失敗時は数分待って再試行してください。

**Q：日報を間違えて提出した**  
A：日報ページまたはカレンダーから鉛筆アイコンで修正できます（自分の日報のみ。管理者は全員分修正可）。

**Q：案件を誤って作成してしまった**  
A：案件は論理削除（非表示）のみ対応しています。誤作成の場合は管理者に連絡してください。

### 操作・システムに関する問い合わせ

システムの不具合・操作不明点は 平等 久盛 まで連絡してください。
