### 現場経費内訳の対応（Excel右側枠）

Excel では現場経費を以下の6項目で管理。パーセンテージは「工事価格に対する比率」。

| Excel項目 | 対応 | 比率設定 |
|---|---|---|
| 契約印紙代 | 手動入力 | 定額 |
| 労災保険料 | 手動入力 | 定額 |
| 工事保険料 | 手動入力 | 定額 |
| 事務用品費 | 率×自動計算 | 0.05% |
| 通信交通費 | 率×自動計算 | 0.3% |
| 雑費 | 率×自動計算 | 0.15% |

→ QCDS の `qcds_expense_items`（site_overhead カテゴリ）から自動取得可能。「修正しないでください」注記はUIでロックボタン付き表示で対応。

| 種別 | 背景色 | ボーダー | ラベル |

### G-1：DBスキーマ（Alembicマイグレーション）

```sql
-- 工事台帳 手動入力補完テーブル（新規）
CREATE TABLE project_ledger_meta (
  id                      UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  project_id              UUID NOT NULL UNIQUE REFERENCES projects(id) ON DELETE CASCADE,
  original_client_name    VARCHAR(255) NULL,    -- 元発注者
  work_summary            TEXT NULL,            -- 工事概要・情報経緯・発注者要望事項（統合テキスト）
  information_history     TEXT NULL,            -- 情報経緯（別欄の場合）
  client_requirements     TEXT NULL,            -- 発注者要望事項
  prev_construction_self  BOOL NULL,            -- 前施工: 当社
  prev_construction_year  SMALLINT NULL,        -- 当社施工年
  prev_construction_other VARCHAR(255) NULL,    -- 他社施工者名
  target_profit_rate      DECIMAL(5,2) NULL,    -- 目標営業利益率(%) ← Excel「目安」欄
  target_profit_amount    DECIMAL(12,0) NULL,   -- 目標営業利益額（rate から自動計算も可）
  created_at              TIMESTAMPTZ DEFAULT now(),
  updated_at              TIMESTAMPTZ DEFAULT now()
);

-- projects テーブルに実施工期カラム追加
ALTER TABLE projects ADD COLUMN actual_start DATE NULL;
ALTER TABLE projects ADD COLUMN actual_end   DATE NULL;

-- qcds_direct_works テーブルに取決済チェック追加
ALTER TABLE qcds_direct_works ADD COLUMN agreed_confirmed BOOL DEFAULT FALSE;

-- 工事台帳承認テーブル（新規）
-- ※承認枠はExcel準拠で4枠: 社長・建築部長・経理・担当
CREATE TABLE ledger_approvals (
  id             UUID PRIMARY KEY DEFAULT gen_random_uuid(),
  project_id     UUID NOT NULL REFERENCES projects(id) ON DELETE CASCADE,
  role_label     VARCHAR(30) NOT NULL,  -- '社長'|'建築部長'|'経理'|'担当'
  approver_id    UUID NULL REFERENCES users(id),
  approved_at    TIMESTAMPTZ NULL,
  comment        TEXT NULL,
  display_order  SMALLINT NOT NULL DEFAULT 0,
  created_at     TIMESTAMPTZ DEFAULT now(),
  UNIQUE (project_id, role_label)
);

-- 初期データ: 全案件に4枠分のレコード挿入（マイグレーション時）
INSERT INTO ledger_approvals (project_id, role_label, display_order)
SELECT id, role, row_number() OVER ()
FROM projects, (VALUES ('社長',0),('建築部長',1),('経理',2),('担当',3)) AS r(role, ord)
WHERE projects.deleted_at IS NULL;
```
