06 · 原則

このプロダクトで大切にすること

数字を間違えない
金額は大きく・等幅・必ず3桁カンマ。確認ステップを挟む。
「承認」は重い行為
印影=責任。承認ボタンは派手にせず、必ず内容確認を促す。
Excelに違和感がない密度
テーブルは詰めめ、フィールドはグリッド整列、Tab/Enterで移動。
現場とオフィスの感覚差
PCは情報密度MAX、モバイル（PWA）は閲覧・進捗・写真に集中。
